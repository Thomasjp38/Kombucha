6) Manual Actuator Controls

Set the oxygen run/wait timing in the Parameters / Thresholds card, then enable the timer here.
If a latch/lockout is active and you try to force an actuator ON, this page blocks the command and shows a “resolve alert first” popup. The firmware still enforces the real hardware lockout.
